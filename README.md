# BoostyBridge

## ru

**BoostyBridge** - плагин для интеграции Minecraft-сервера с платформой Boosty.  
Позволяет автоматически выдавать привилегии и другие награды игрокам за подписку на Boosty.

В данный момент плагин находится в разработке.  
Актуальная версия разработки доступна в ветке `dev`.

---

## en

**BoostyBridge** is a plugin for integrating a Minecraft server with the Boosty platform.  
It allows you to automatically grant privileges and other rewards to players for their Boosty subscription.

The plugin is currently under development.  
The latest development version is available in the `dev` branch.
